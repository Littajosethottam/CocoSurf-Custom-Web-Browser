# CocoSurf-Custom-Web-Browser
  CocoSurf Custom Web Browser is a personalized Chromium-based web browser designed for seamless, private, and efficient web browsing. 
Python and Html coding is used .The Python code handles the browser’s features, customization options, and interaction with the operating system, while HTML and related web technologies are employed to present web pages and UI elements effectively.
# Features:
    Chromium-based rendering for modern web compatibility

    Customizable interface built with Python and HTML

    Privacy-focused tools and settings

    Lightweight and efficient performance

    Easy to extend with plugins or UI modifications
# Technology Stack
    Python — Core application logic and GUI handling (using PyQt5 or similar)

    HTML / CSS / JavaScript — Web content rendering and UI elements

    Chromium engine — Embedded for secure and fast web page display
# Usage
Launch the application to start browsing. Customize settings through the options menu to adjust privacy, appearance, and extensions.
